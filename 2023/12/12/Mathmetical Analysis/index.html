0. Real Number Theory

construction of real number set

Dedekind cuts \(\mathbb{R}\) is all \(\alpha\) such that:

  • \(\alpha\neq\varnothing\) and \(\alpha\neq\mathbb{Q}\)
  • \(\forall p\in\alpha,\forall q\in\mathbb{Q},(q<p\to q\in\mathbb{Q})\)
  • \(\forall p\in\alpha,\exists r\in\alpha,(p<r)\)

define the order on \(\mathbb{R}\): \((\mathbb{R},\leq)\) define the operator on \(\mathbb{R}\): \((\mathbb{R},+,\times)\)

Cauchy sequence

density \(\forall x_1,x_2\in\mathbb{R},\exists x\in\mathbb{R},(x_1<x<x_2)\)

Completeness

the completeness axiom of the reals - every nonempty subset of real numbers that is bounded above has a supremum - any Cauchy sequence converges

1. Limit Theory

Definition of Limits

Limits of Sequences

Set an infinite sequence \(\{x_n\}\), if: \[\exists L,\forall\epsilon>0,\exists N\in\mathbb{N},\forall n,(n>N\to\lvert x_n-L\rvert <\epsilon)\tag1\] we say that it converge to \(L\), which is denoted as \(\lim_{n\to\infty}x_n=L\)

\[\forall G>0,\exists N\in \mathbb{N},\forall n,(n>N\to x_n>G)\tag2\] we say that it is disvergent, which is denoted as \(\lim_{n\to\infty}x_n=\infty\)

Limits of Functions

Set a function \(f(x)\), suppose \(\mathring{U}(x_0)\subseteq D_f\), if: \[\exists L,\forall\epsilon>0,\exists\delta>0 ,\forall x,(\lvert x-x_0 \rvert<\delta\to \lvert f(x)-L \rvert<\epsilon)\tag1\] we say that it converge to \(L\), which is denoted as \(\lim_{x\to x_0}f(x)=L\)

\[\exists L,\forall\epsilon>0,\exists G\in R ,\forall x,(x>G\to \lvert f(x)-L \rvert<\epsilon)\tag2\] we say that it converge to \(L\), which is denoted as \(\lim_{x\to\infty}f(x)=L\)

\[\forall K>0,\exists\delta>0 ,\forall x,(\lvert x-x_0 \rvert<\delta\to f(x)>K)\tag3\] we say that it is disvergent, which is denoted as \(\lim_{x\to x_0}f(x)=\infty\)

\[\forall K>0,\exists G>0,\forall x,(x>G\to f(x)>K)\tag4\] we say that it is disvergent, which is denoted as \(\lim_{x\to\infty}f(x)=\infty\)

One-sided Limits

  • left-sided limit: \(\lim_{x\to x_0^+}f(x)\)
  • right-sided limit: \(\lim_{x\to x_0^-}f(x)\)

Relationship

subsequence: \(\lim_{x\to x_0}f(x)=L \leftrightarrow \forall \{x_{n_k}\}(\lim_{k\to \infty}x_{n_k}=L)\)

Heine theorem \[\lim_{x\to x_0}f(x)=L\iff\forall\{x_n\},(\lim_{n\to\infty}x_n=x_0\to\lim_{n\to\infty}f(x_n)=L)\]

Applying Limits

Properties of Limits

  • uniqueness of limit(唯一性)
    • \((\lim_{x\to x_0}f(x)=L_1)\wedge(\lim_{x\to x_0}f(x)=L_2)\to (L_1=L_2)\)

    • \((\lim_{x\to x_0}f(x)=L)\to(\exists M,\forall x\in\mathring{U}(x_0,\delta),(M\geq x))\)
  • locally sign-preserving property(保号性)

Infinitesimal

Infinitesimals

if ==\(\lim \alpha(x)=0\)==, then we call \(\alpha(x)\) is the infinitesimal quantity in this limit process

a theorem that relate limits and infinitesimal quantities: \[\lim f(x)=L\iff f(x)+\alpha(x)=L,\lim\alpha(x)=0\]

properties of infinitesimal: \(\lim C\cdot\alpha=0\) based on infinitesimals, we can deduce the four arithmetic rules of limits

Order of infinitesimals

sometimes the four arithmetic rules of limits fail, we call these limits indeterminate: examples: \(\cfrac 00,\cfrac\infty\infty,0\cdot\infty,\infty-\infty,\infty^0,0^0,(1+0)^\infty\) Fortunately, all above can be reduced to the the form of \(\cfrac 00\)

  • Higher-order infinitesimal: \(\lim\cfrac\beta\alpha=0\) , denoted as ==\(\beta=o(\alpha)\)==
  • Lower-order infinitesimal: \(\lim \cfrac \beta \alpha=\infty\) , denoted as ==\(\alpha=o(\beta)\)==
  • Same-order infinitesimals: \(\lim \cfrac \beta \alpha=c\) , denoted as ==\(\beta\sim c\alpha\)==
  • \(k\)th-order infinitesimal: \(\lim \cfrac \beta {\alpha^k}=c\) , denoted as ==\(\beta\sim c\alpha^k\)==
  • Equivalent infinitesimal: \(\lim \cfrac \beta \alpha=1\) , denoted as ==\(\beta\sim\alpha\)==

Among all, Equivalent infinitesimal is the most important one, for the reason that we use them to simplify equations very often, which is based on the equivalent infinitesimal substitution theorems: If \(\alpha\sim\beta\) , then \(\lim \alpha f(x)=\lim \beta f(x)\)

after we determine a Reference infinitesimal, often \(x\), we look for \(k\) such that \(f(x)=cx^k+o(x^k)\), then \(cx^k\) is called Infinitesimal principal

Continuity

Definition of Continuity

Continuity at one point set a function \(f(x)\), suppose \(\mathring{U}(x_0)\subseteq D_f\), if limit below exists: \[\lim_{x\to x_0}f(x)=f(x_0)\] then we said \(f(x)\) is continuous at \(x_0\)

one-sided continuity:

  • left-sided continuity: \(\lim_{x\to x_0^+}f(x)= f(x_0)\)
  • right-sided continuity: \(\lim_{x\to x_0^-}f(x)= f(x_0)\)

Continuity on an interval If \(\forall x_0\in (a,b),(\lim_{x\to x_0}f(x)=f(x_0))\) and \(\lim_{x\to a^+}f(x)= f(a)\) , we say that \(f(x)\) is continuious on \([a,b)\) , denoted as \(f(x)\in C[a,b)\) distinction between open interval & closed interval

Discontinuity

  • Removable Discontinuity: \(\lim_{x\to x_0}f(x)=L\) but \(L\neq x_0\)
  • Jump Discontinuity: \(\lim_{x\to x_0^+}f(x)\neq\lim_{x\to x_0^-}f(x)\)
  • Infinite Discontinuity: \(\lim_{x\to x_0}f(x)=\infty\)
  • Oscillatory Discontinuity: \(\lim_{x\to0}\sin\frac 1x\)

uniform continuity

Properties of continuous function

  • continuous functions remain continuous after basic arithmetic operations, composite operations, and inverse operations

  • elementary functions are all continuous

  • continuous function on a closed interval

    • has both minimum and maximum
    • intermediate value thm \[f(x)\in C[a,b]\to \forall c\in (f(a),f(b)),\exists\xi\in(a,b),(c=f(\xi))\]

2. Derivative & Differential

Derivative

Definition of Derivative

set a function \(f(x)\), suppose \(\mathring{U}(x_0)\subseteq D_f\), the derivative of \(f(x)\) at \(x_0\) is defined as: \[f'(x_0)=\lim _{\Delta x\to 0}\cfrac {f(x_0+\Delta x)-f(x_0)}{\Delta x}\] if the limit exists, we say that \(f(x)\) is derivable at \(x_0\)

derivative function of \(f(x)\) is denoted as \(f'(x)\) and it is a function such that: \[\forall x\in D_f,(f'(x)=\lim _{\Delta x\to 0}\cfrac {f(x_0+\Delta x)-f(x_0)}{\Delta x})\]

one-sided derivative:

  • right-sided derivative: \(f'_+(x)\)
  • left-sided derivative: \(f'_-(x)\)

Derivability

\(f\) is continuous \(\Rightarrow\) \(f\) is derivable continue but not derivable: two cases where derivative limit does not exist

  • cusp: \(f'_+(x)\neq f'_-(x)\)
  • vertical: \(f'(x)=\infty\)

derivative of a derivable function is continue or has oscillatory discontinuity

3. Taylor Formula

Differential Mean Value Theorem

Differential Mean Value Theorem reflects the relationship between functions and derivatives

Fermat's Extreme Value Theorem Suppose \(f\in D\mathring{U}(x_0)\) and \(\forall x\in \mathring{U}(x_0),\bigg(f(x_0)\leq f(x)\bigg)\) , than \(f'(x_0)=0\)

Rolle mean value theorem Suppose \(f\in C[a,b]\), \(f\in D(a,b)\), and \(f(a)=f(b)\), then: \[\exists\xi\in(a,b),\bigg(f'(\xi)=0\bigg)\] >[!note] Proof >if \(f\) is a constant function, then \(\forall x\in(a,b),(f'(x)=0)\) >otherwise, it has an extremum, then use Fermat Thm

Lagrange mean value theorem Suppose \(\big(f,g\in C[a,b]\big)\wedge \big(f,g\in C^1(a,b)\big)\), then: \[\exists \xi \in (a,b),\bigg(f'(\xi)=\cfrac {f(a)-f(b)}{a-b}\bigg)\] >[!note] Proof >constructing auxiliary function \(\phi (x)=f(x)-\cfrac {f(a)-f(b)}{a-b}x\) >according to Rolle's Thm, \(\exists\xi\in(a,b),\bigg(\phi'(\xi)=0\bigg)\)

Cauchy mean value theorem Suppose \(\big(f,g\in C[a,b]\big)\wedge \big(f,g\in C^1(a,b)\big)\wedge \big(g'(x)\neq0\big)\), then: \[\exists\xi\in(a,b),\bigg(\cfrac {f'(\xi)}{g'(\xi)}=\cfrac {f(a)-f(b)}{g(a)-g(b)}\bigg)\]

[!note] Proof constructing auxiliary function \(\phi (x)= f(x)-\cfrac{f(a)-f(b)}{g(a)-g(b)}g(x)\) according to Lagrange mean value Thm, \(\exists\xi\in(a,b),\bigg(\phi'(\xi)=0\bigg)\)

L'Hôpital's rule

Suppose \(\big(f,g\in C^1\mathring{U}(x_0,\delta)\big)\wedge\big(\lim _{x\to x_0}f(x)=\lim _{x\to x_0}{g(x)}=0\big)\wedge\big(g'(x_0)\neq0\big)\), then: \[\lim _{x\to x_0}\cfrac {f(x)}{g(x)}=\lim _{x\to x_0}\cfrac {f'(x)}{g'(x)}\]

Proof WLOG let \(f(x_0)=g(x_0)=0\) By Definition of Derivative: \(\lim _{x\to x_0}\cfrac {f(x)}{g(x)}=\lim_{x\to x_0}\cfrac{\cfrac{f(x)-f(x_0)}{x-x_0}}{\cfrac{g(x)-g(x_0)}{x-x_0}}=\lim _{x\to x_0}\cfrac {f'(x)}{g'(x)}\) Use Cauchy mean value theorem: \(\exists\xi\in(x,x_0),\big(\cfrac{f'(\xi)}{g'(\xi)}=\cfrac{f(x)-f(x_0)}{g(x)-g(x_0)}\big)\) , so \(\lim _{x\to x_0}\cfrac {f(x)}{g(x)}=\lim_{\xi\to x_0}\cfrac{f'(\xi)}{g'(\xi)}=\lim _{x\to x_0}\cfrac {f'(x)}{g'(x)}\)

Proof: let \(\Phi(x)=\int_a^xf(x)\mathrm{d}x\), then \(\Delta\Phi(x)=\Phi(x+\Delta x)-\Phi(x)=\int_x^{x+\Delta x}f(x)\mathrm{d}x\) \(\Rightarrow\exists\xi\in(x,x+\Delta x),\Big(\int_x^{x+\Delta x}f(x)\mathrm{d}x=f(\xi)\Delta x\Big)\) \(\Rightarrow\Delta x\to0,\Big(\int_x^{x+\Delta x}f(x)\mathrm{d}x=f(x)\Delta x\Big)\) \(\Rightarrow\lim_{\Delta x\to 0}\cfrac{\Phi(x+\Delta x)-\Phi(x)}{\Delta x}=f(x)\)
