Set Theory

Set

notations

  • relashionship of set
    • \(A\subseteq B\iff \forall x(x\in A\to x\in B)\)
    • \(A=B\iff \forall x(x\in A\leftrightarrow x\in B)\)
  • operations of set
    • \(A\cap B=\{ x|x\in A \wedge x\in B\}\)
    • \(A\cup B=\{ x|x\in A \vee x\in B\}\)
    • \(A\setminus B=\{x|x\in A\wedge x\notin B\}\)
    • \(A^c=\{x|x\notin A\}\)
    • \(A\Delta B=(A\cup B)\setminus(A\cap B)\)
    • \(\bigcap_{i\in I}A_i=\{x|\forall i\in I,x\in A_i\}\)
    • \(\bigcup_{i\in I}A_i=\{x|\exists i\in I,x\in A_i\}\)

limit of set

a sequence of set \(\{A_n\}_{n=1}^\infty\) if \(A_n\subset A_{n+1}\), we says it is monotone increasing and denote \(\bigcup_{n=1}^\infty A_n\) as \(\lim_{n\to\infty}A_n\) if \(A_n\supset A_{n+1}\), we says it is monotone decreasing and denote \(\bigcap_{n=1}^\infty A_n\) as \(\lim_{n\to\infty}A_n\)

\(\{\bigcap_{k=n}^\infty A_k\}_{n=1}^\infty\) is monotone increasing \[\lim\inf_{n\to\infty} A_n=\bigcup_{n=1}^\infty\bigcap_{k=n}^\infty A_k\] \(x\in\lim\inf_{n\to\infty}A_n\iff\exists n,\forall k\geq n,(x\in A_k)\) \(\{\bigcap_{k=n}^\infty A_k\}_{n=1}^\infty\) is monotone decreasing \[\lim\sup_{n\to\infty} A_n=\bigcap_{n=1}^\infty\bigcup_{k=n}^\infty A_k\] \(x\in\lim\sup_{n\to\infty}A_n\iff\forall n,\exists k\geq n,(x\in A_k)\)

n-tuple: \[(x_1,x_2,\cdots,x_n)=\{x_1,\{x_1,x_2\},\{x_1,x_2,x_3\},\cdots\}\] Cartesian product \[A\times B=\{(a,b)|a\in A \wedge b\in B\}\] specifically, \(A^4=A\times A\times A\times A\)

Power set the power set of \(S\), denoted as \(\mathscr{P}(S)\), such that \[X\in\mathscr{P}(S)\iff X\subseteq S\]

Partition \(\mathscr{A}\) is a partition of nonempty set \(X\) if - \(\forall A\in\mathscr{A},(A\neq\emptyset)\) - \(\bigcup_{A\in\mathscr{A}}A=X\) - \(\forall A_1,A_2\in X,(A\cap B\neq\emptyset\to A_1=A_2)\) ### mapping \(f\) is a mapping from \(X\) to \(Y\), denoted as \(f:X\to Y\),if - \(\forall x\in X,\exists y\in Y,\Big((x,y)\in f\Big)\) - \(\forall x\in X,\forall y_1,y_2\in Y,\Big((x,y_1)\in f\wedge(x,y_2)\in f\to y_1=y_2\Big)\)

\(f\) is a correspondence rule between two nonempty set \(X,Y\) and each \(x\in X\) map to a unique \(y\in Y\), then \(f\) is a mapping from \(X\) to \(Y\) denoted as \(f:X\to Y\) - domain: \(D=X\) - codomain: \(Y\) - range: \(R=\{y|y=f(x),x\in D\}\) - If \(x\) is an element of \(X\), and \(f(x)=y\), then \(x\) is preimage of \(y\), \(y\) is image of \(x\), denoted as \(x\mapsto y\)

bijection & injection & surjection - injection/one to one: \(\forall x_1,x_2\in X,(f(x_1)=f(x_2)\to x_1=x_2)\) - surjection/onto: \(\forall y\in Y,\exists x \in X,(f(x)=y)\) - bijection: \(\forall x_1,x_2\in X,(f(x_1)=f(x_2)\to x_1=x_2)\wedge \forall y\in Y,\exists x \in X,(f(x)=y)\)

  • function
    • a mapping from number set to number set
  • functional
    • a mapping from function set to number set
  • operator
    • a mapping from function set to function set
  • identity function
    • \(i_X:X\to X\) defined by \(i_X(x)=x\)
  • composition
    • \(f:A\to B,g:B\to C\), then \(g\circ f(x)=g(f(x))\)
    • \(h\circ (g\circ f)=(h\circ g)\circ f\)
  • inverse
    • let \(f:X\to Y\) a bijection, \(f(x)=y\iff f^{-1}(y)=x\)
    • \(f^{-1}\circ f=i_X\) and \(f\circ f^{-1}=i_Y\)
    • \((f\circ g)^{-1}=g^{-1}\circ f^{-1}\)
  • image of \(A\subseteq X\)
    • \(f(A)=\{f(x)|x\in A\}\)
  • inverse images of \(B\subseteq Y\)
    • \(f^{-1}(B)=\{y|f(y)\in B\}\)
  • transformation \(f:X\to X\)
    • permutation \(T\) is a transformation and a bijetion
    • identity map \(I:X\to X,\forall x\in X,(I(x)=x)\)

indicator function suppose \(S\subseteq X\), then \(\chi_S:X\to\{0,1\}\) defined by \(\chi_S(x)=0\) if \(x\notin S\), and \(\chi_S(x)=1\) if \(x\in S\)

Relation

if \(R\subseteq X^n\), it is called n-ary relation more often, we discuss binary relation

suppose \(R\) is a binary relation on \(X\), it has the following properties conditionally - reflexive property - \(\forall x\in X((x,x)\in R)\) - symetric property - \((x_1,x_2)\in R\to(x_2,x_1)\in R\) - antisymetric property - \((x_1,x_2)\in R\wedge (x_2,x_1)\in R\to x_1=x_2\) - strongly antisymetric property - \((x_1,x_2)\in R\to(x_2,x_1)\not\in R\) - transitive property - \((x_1,x_2\in R\wedge(x_2,x_3)\in R\to(x_1,x_3)\in R)\) - EXAMPLE1:"greater than" relation on the set of real numbers is transitive - 传递闭包(transitive closure) - The transitive closure of \(R\), denoted by \(R^*\),is defined as \(R^*=R\bigcup\{(x_i,x_j)|\exists x_{k_1}\cdots x_{k_t}((x_i,x_{k_1}),(x_{k_1},x_{k_2}),\cdots,(x_{k_t},x_j)\in R)\}\) - 其实\(R^*\)就是包含\(R\)的具有传递性的最小集合 - Warshall Algorithm 寻找传递闭包的算法 - n次循环{找到(a,i)(i,b),add \(A\times B\)}

\(R\) is an equivalence relation if and only if it is: - reflexive - symetric - transitive when \(R\) is an equivalence relation set, we write \((x,y)\in R\) as \(x\sim y\), we also define \(E_x=\{y\in X:x\sim y\}\), called equivalence class

preorder

for a given \((X,R)\), \(X\) is a ==preordered set==, a.k.a. quasi-order set, when \(R\) is: - irreflexive ? reflexive - transitive

partial order

for a given \((X,R)\), \(X\) is a ==partially ordered set==, when \(R\) is: - reflexive - antisymetric - transitive

we write \((x,y)\in R\) as \(x\preceq y\), called \(x\) precede \(y\) if \(x\preceq y \wedge x\neq y\), they we write it as \(x\prec y\), called \(x\) strictly precede \(y\)

==comparability== suppose \(x_1,x_2\in X\) and \(X\) is a partially ordered set, then if \(x_1\preceq x_2\vee x_2\preceq x_1\), we says they are comparable, otherwice, they are uncomparable and written as \(x_1||x_2\) trichotomy of relations: \[x_1\preceq x_2\vee x_2\preceq x_1\vee x_1||x_2\]

given a partial order set \((X,\preceq)\) and a set \(S\subseteq X\) - lower bound & upper bound: - if \(\exists m\in X,\forall x\in S,(m\preceq x)\), then \(m\) is a lower bound of \(S\) - infimum & supremum: - if \(l\) is a lower bound of \(S\), and \(\forall m,(m\preceq l)\), then \(l\) is an infimum of \(S\) - \(l\) is denoted as \(\inf S\) - minimal & maximal: - if \(\exists m\in S,\forall x\in S,(m\npreceq x)\) , then \(m\) is a minimal of \(S\) - minimum & maximum: - if \(\exists m\in S,\forall x\in S,(m\preceq x)\) then \(m\) is a minimum of \(S\)

==Hasse diagram== we say \(x\) is a immediate predcessor of \(y\) if and only if \[(x\prec y)\wedge(\neg\exists z\in X,(x\prec z\wedge z\prec y))\] we denote it as \(x\ll y\)

then, we draw a line between two elements if and only if one is an immediate predcessor of the other, this is called Hasse diagram ![[Pasted image 20231108153940.png|250]]

totally ordered set

a partially ordered set \(X\) is a ==totally ordered set==, a.k.a. chain or linear order, when: \[\forall x,y\in X,(x\preceq y\vee y\preceq x\vee x=y)\] \(A\) is a ==well-ordered set==, when: \[\forall S\subset X,S\neq\emptyset,\big(\exists x\in S,\forall y\in S,(x=y\vee xRy)\big)\]

we can deduce idempotent law then: \(x\wedge x=x\)
